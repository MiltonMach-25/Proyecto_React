# Proyecto Relaxify - Frontend (React)

## Descripción
**Relaxify** es una aplicación enfocada en el bienestar emocional y la relajación.  
Esta parte del proyecto corresponde al **Frontend**, desarrollado con **React + Vite**.

El sistema permite:
- Registro e inicio de sesión de usuarios (simulado en frontend con almacenamiento local).
- Visualización de una interfaz limpia y moderna.
- Navegación entre las secciones principales de la plataforma Relaxify.

> Nota: El sistema está preparado para conectarse con un backend en Node.js y una base de datos MySQL, aunque para la versión actual de presentación académica el manejo de datos se realiza desde el navegador (almacenamiento local), ETC.

---

## Tecnologías Utilizadas
- React 18
- Vite 7
- JavaScript (ES6+)
- CSS modular
- ESLint para control de calidad del código
