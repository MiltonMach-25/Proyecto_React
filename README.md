# Relaxify – Actualización del Frontend (React + Vite)

## 1. Descripción general

Este repositorio contiene la parte correspondiente al **Frontend del Proyecto Relaxify**, una aplicación enfocada en ofrecer herramientas de relajación, bienestar emocional y experiencias interactivas a través de una interfaz amigable.

La aplicación está desarrollada con **React** utilizando **Vite** como entorno de desarrollo, lo que permite una carga rápida y un flujo eficiente durante la programación.

El frontend incluye:

- Formulario de registro de usuario  
- Pantalla de verificación mediante código  
- Panel o dashboard principal  
- Sección de meditaciones y contenido de bienestar  
- Seguimiento básico del progreso del usuario  
- Integración de imágenes, audios y material visual desde la carpeta `assets`

Este frontend está preparado para conectarse con un backend en Node.js.  
En la versión presentada para el proyecto académico, los datos se manejan de forma temporal desde la aplicación local.

## 3. Estado actual del proyecto

La estructura del frontend está completamente organizada y funcional para uso académico.  
El sistema permite:

- Registrar usuarios  
- Enviar datos al backend  
- Permitir verificación por código  
- Navegar por las distintas secciones  
- Integrar contenidos multimedia  

El proyecto está preparado para ampliarse y conectarse con MySQL en una versión posterior.
